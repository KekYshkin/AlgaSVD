# Исследование псевдообратной матрицы Мура-Пенроуза

Данный проект посвящен изучению математических свойств псевдообратной матрицы и её применению в задачах восстановления сигналов и анализа зашумленных систем.

## Описание проекта
Программа демонстрирует работу с недоопределенными системами уравнений ($m < n$) и визуализирует геометрию решений в 3D. 

## Требования
Для запуска необходим Python 3.x и следующие библиотеки:
* `numpy`
* `matplotlib`

Установка:
```bash
git clone https://github.com/KekYshkin/AlgaSVD.git

pip install -r requrements.txt
```

## Как запустить
Просто запустите основной скрипт:
```bash
python main.py
```

## Результаты работы
Программа выводит в консоль результаты проверок математических свойств и открывает два окна с графиками:
1. **Логарифмические графики** зависимости ошибки от шума и обусловленности.
2. **3D-сцена**, показывающая истинный вектор, проекцию на ядро (часть, которая теряется) и восстановленный вектор (проекция минимальной нормы).
